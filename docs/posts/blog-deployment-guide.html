博客文章部署完整指南

發佈於 2025-07-30 22:11:05
在創建新文章時，請遵循以下格式： 第一行：文章標題 第二行：空白行 第三行開始：文章內容 文章檔案應保存為 `.txt` 格式，並放置在 `content/` 目錄中。 步驟1: 構建網站 python build.py 步驟2: 檢查Git狀態 git status 步驟3: 添加所有更改 git add . 步驟4: 提交更改 git commit -m "新增文章: 我的新文章" 步驟5: 推送到GitHub git push origin main 部署後驗證 訪問 https://co2sou.github.io/blog
