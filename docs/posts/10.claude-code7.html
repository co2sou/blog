三）
編譯 Windows exe 的完整步驟：

1. 安裝 Rust：
- 訪問 https://rustup.rs/
- 下載並運行 rustup-init.exe
- 按默認設置安裝
2. 安裝 Visual Studio Build Tools：
- 訪問 https://visualstudio.microsoft.com/visual-cpp-build-tools/
- 下載並安裝（選擇 C++ 構建工具）
3. 編譯：
bun run tauri build
4. 編譯完成後，exe 文件將位於：
src-tauri/target/release/claudia.exe
